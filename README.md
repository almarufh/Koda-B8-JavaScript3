## OBJECT FUNCTION LINGKARAN

# Cara mengugunaka

untuk menggunakan object function ini panggil variabel object lingkaran dan masuk kedalam properti yang berisi peroperti Luas ataupun Keliling yang berisikan function dengan menggunakan 1 parameter yang wajib dalam bentuk Number, jika bukan number maka akan m,uncul peringatan "Input must be number" jika benar maka akan mengoutputkan nilai dengan tipe data Number
